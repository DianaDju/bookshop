# Bookshop Project
Интернет-магазин книг с использованием Google Books API.

### Стек:
* Webpack 5 (сборка, минификация, asset modules)
* SCSS (модульная структура, @use)
* JavaScript (ES6 Modules, LocalStorage)
* API: Google Books

### Как запустить:
1. `npm install`
2. `npm run start` (разработка)
3. `npm run build` (сборка)
